Teachable Machine

Teachable Machine es una herramienta web que permite crear un modelo de aprendizaje automático, el cual mediante varios ejemplos dados por el usuario va a procesar la información entregada y con esto diseñara el modelo.

Gracias a este sitio web las personas pueden llegar a comprender de mejor manera el Machine Learning y la Inteligencia Artificial. De una manera muy fácil se pueden crear modelos que muestren las poses, sonidos o imágenes que presenten los usuarios.

Red Neuronal

Una Red Neuronal es una manera de enseñarle a la computadora a procesar datos mediante Inteligencia Artificial usando como referencia el cerebro humano. Esto permite que aprenda automticamente a corregir sus errores.

Permiten que los programas tomen mejores desiciones y sean mas optimos, mejorando la calidad del software y su rendimiento.
